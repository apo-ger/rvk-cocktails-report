Project Overview

Transforming Iceland's premier cocktail service from manual operations to intelligent automation.

🍸 The Company

Reykjavík Cocktails delivers premium cocktail experiences across Iceland, from intimate private events to large corporate gatherings. With four specialized divisions—Kitchen operations, RVK Cocktails service, Vinskolin, and Cocktailskolin—the business handles complex logistics daily.

Each event requires precise coordination: ingredient sourcing, recipe calculations, equipment preparation, staff scheduling, and client communication—all traditionally managed through manual, time-intensive workflows.

⏰ The Challenge

Manual workflows consume hours of valuable time that could be spent on creative cocktail development and exceptional client service. Inventory tracking relies on spreadsheets, recipe calculations are done by hand, and preparation reports require tedious manual compilation.

These time-intensive processes create bottlenecks, increase error rates, and limit the business's ability to scale efficiently while maintaining the premium quality servive.

💡 The Solution: Intelligent Operations Platform

The proposed platform transforms manual, time-intensive workflows into intelligent, automated processes. From gig creation to mise en place calculations, inventory management to team coordination—every operational aspect is streamlined for efficiency and precision.

Technology Stack

The platform leverages enterprise-grade technologies that power companies like Netflix, Airbnb, and Uber. Our carefully curated stack ensures exceptional performance, unlimited scalability, and rock-solid reliability for years to come.

⚛️

Next.js 15

React 19 Framework with App Router

Next.js provides the foundation for our application, offering server-side rendering, automatic code splitting, and optimized performance out of the box. Version 15 with React 19 brings improved performance and the latest React features including enhanced server components.

  • Built-in performance optimizations with automatic image optimization
  • Server Components for reduced client-side JavaScript
  • Incremental Static Regeneration for dynamic content
  • API routes for backend functionality
  • Excellent developer experience with Fast Refresh
🗄️

MongoDB

Document Database with Mongoose ODM

MongoDB's document-based structure perfectly matches our data model where cocktails, events, and inventory items have varying attributes. This flexibility is crucial for a business where a cocktail might have 3 ingredients or 12, plus variable garnishes and preparation notes.

  • Flexible schema design adapts to changing needs & requirements
  • Native support for nested documents (perfect for event details)
  • Powerful aggregation framework for analytics
  • Built-in geospatial queries for location-based features
  • Change streams for real-time updates
🔐

NextAuth.js v4

Complete Authentication Solution

NextAuth.js provides a complete authentication solution with support for multiple providers, database sessions, and JWT tokens. It handles the complex security requirements while remaining flexible for custom needs.

  • Secure by default with CSRF protection
  • Support for OAuth providers (Google, etc.)
  • Database sessions for enhanced security
  • Role-based access control (RBAC) support
🗺️

Mapbox GL

Interactive Map Visualization

Mapbox GL provides elegant, interactive maps showing all gig locations across Iceland, with custom markers and real-time filtering capabilities.

  • Vector maps with smooth zooming
  • Custom styling
  • Clustering for multiple locations
  • Mobile gesture support
  • Offline map caching potential
📅

FullCalendar

Professional Calendar Component

FullCalendar powers our event scheduling system with a feature-rich, interactive calendar that integrates smoothly with our gig management workflow.

  • Multiple view modes (month, week, day, list)
  • Mobile-responsive design
  • Real-time updates
🔧

Zustand

Lightweight State Management

Zustand provides a solid state management solution that's perfect for our inventory and gig management needs. It offers excellent performance with minimal boilerplate, making the application responsive and maintainable.

  • Lightning-fast state updates with automatic re-renders
  • Simple API that's easy to understand and maintain
  • Persist state to localStorage for offline capabilities
  • DevTools support for debugging state changes
🔄

React Query

Powerful Data Fetching

React Query (TanStack Query) handles all server state management, providing automatic caching, background refetching, and optimistic updates for a snappy user experience.

  • Automatic request deduplication
  • Background data synchronization
  • Optimistic updates for instant feedback
  • Built-in error and loading states
  • Offline support with cache persistence
🎨

CSS Modules

Component-Scoped Styling

CSS Modules provide locally-scoped CSS by default, preventing style conflicts and making our components truly modular. Combined with CSS custom properties, we achieve a powerful theming system that supports light/dark modes seamlessly.

  • Component-level style isolation
  • Custom properties enable dynamic theming
  • Smaller bundle size - only ship what you use
  • Native browser support
